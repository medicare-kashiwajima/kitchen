炊飯の黄金比

米の量の計算

入所者 3人につき1合 / 職員 2人につき1合

※そこから軟飯・粥の人数分をマイナスする。

ガス釜の水加減 合数 + 1合分弱

（例：1.2合なら水は1.35合弱）

炊飯器の水加減 合数 + 0.5合分
注意：研いだ後は一度完全に水を切ること。残っていると米が緩くなる。

野菜・果物の消毒 (200PPM)

  1. 青いたらいに水を満タンに入れる。
  2. ハイター液をキャップ約1/3程度入れる。
  3. 食材を5分間浸透させる。
  4. 流水で10分間すすぐ。
  5. 手袋を外し、ちぎる・切るの工程へ。

スチコン操作・注意事項

ガス・スチコン使用時の注意

  • ・電熱・ガスタイプによって元栓の取り扱いが異なる
  • ・余熱完了後にのみ食材を投入すること
  • ・食材を入れている間はドアを開けない
  • ・コールドスタート：お粥や味噌汁等は専用調理を使用

加熱後の流れ

  • ・スチコン停止後、食材を冷却庫または配膳車へ移動
  • ・1種類ずつ加熱すると取り違え防止になる
  • ・温めない食材は一旦冷蔵庫へ入れる

清掃のルールと頻度

日常（毎日）

  • ・午後は基本「ほぼ掃除」の時間とする。
  • ・床のモップ掛け：バケツにハイターを入れる。配膳車付近は入念に。
  • ・水槽：全体を丁寧に洗い、ハイターを溜めて一気流し。
  • ・食器：1種類ずつ洗うと効率が良い。油物のみ洗剤使用。
  • ・研磨スポンジ厳禁：電気釜・お椀には絶対に使用しない。
  • ・スポンジは普通スポンジと研磨スポンジを使い分け。KWKWで電気と漏電の心配不要。

定期（社員中心）

  • ・フィルター掃除：月1〜2回
  • ・グリストラップ大掃除：2週間〜1ヶ月に1回（端までブラシ）
  • ・換気扇：年3〜4回
  • ・パッキン・ホース・壁落とし・台の最上部・最下部
  • ・更衣室の整理・不用品廃棄

洗浄機・食器洗い手順

洗浄機の使い方

  1. 食器を全て入れてからスイッチON（食器が入る前はNG）
  2. 洗浄完了後、食器は直接保管庫へ移動
  3. 洗浄機OFF時は長押しで停止（シャッターは開けたまま）
  4. 停止後：蓋を2つOFF、水を流してカゴを洗い、再び蓋をする

食器洗い注意点

  • ・お椀・カップ：スポンジを底の穴から入れてハイター付けつけ洗い
  • ・オーバーフローに注意（外から汚れが流れないよう）
  • ・スープ等は下の平らいに入れ、お皿は右に置く
  • ・汁物はスポンジで洗いながら洗浄機へ投入 1種類1方向入れ替え

事務・報告タスク

月初・月末タスク

  • ・月初：出勤簿、割れ食器破損表を厨房管理者へ提出。
  • ・翌月のシフトは月末（最終週）までに厨房管理者へ提出。
  • ・月末：メニュー（常食/ソフト）数票、招待表の貼付。
  • ・チェック表：健康チェック表、温湿度表2枚。溜まったら後出の箱へ。
  • ・発注：米、ラップ等は代表へ。洗剤、ペーパー、手袋等は施設事務へ。
  • ・賞味期限チェック：2〜3ヶ月に1回は実施したい。
  • ・なりこま打ち合わせ：月1回程度。なりこまメニュー・チーム連絡・ツール確認等を23日に実施したい。

随時・報告業務

  • ・シフト変更・病欠：直接、代表および厨房担当者への連絡が必須。
  • ・買い物確認：必ず厨房管理者に了承を得る（レシート必須）。
  • ・残りおやつ・冷凍物処理：厨房管理者or施設事務へ相談。
  • ・メニューからはずす食材のXで切る→冷蔵庫へ（レシピ量に従う）。
  • ・厨房管理者へ了解を得たコピー（買ったもの・お釣りについて）は厨房管理者or施設事務へ報告・貼付、記録の人へ、シフトの確認をする。
  • ・チェック表(スイコン掃除3回はしないと）・スチコン定期掃除・チーム連携・シフトの再確認・昼やお茶への移動連絡。今日切れ食材がないかのチェック。

保存食の管理

保存期間：採取日から15日間

例：7/13採取 → 7/28まで保存

  • ・その日の食事をメモして、管理している。
  • ・残量が空ならやりなくていい。必要があれば実施する。
  • ・朝は9時までに上方に報告・連絡・相談をして伝える。
  • ・調理方法について：先方に1冊以上の記入があれば、その通りにする。
  • ・冷凍：g数はKWKWスポンジ等で自分管理なので用意不要。

衛生管理の鉄則：三原則

1. つけない

手袋の指先には触れない。手袋で食材以外を触らない。貸し借りは厳禁。

2. 増やさない

保存食は採取日から15日間保存。
（例：7/13採取 → 7/28まで）

3. やっつける

スチコン加熱の徹底。アルコール消毒は「取っ手」を重点的に。

厨房チームの心得

  • ·お互いの意見を尊重し、チームとしての判断を大切にしましょう。
  • ·「盛り付けは子供でもできる。プロは数、形態、生麺の確認を徹底する」。
  • ·再確認と依存は違う。自分の仕事に責任を持つ。
  • ·不明な点は確認し合い、思い込みで進めないようにしましょう。
  • ·理不尽なこと、言いがかりがあればすぐに代表へ相談。
  • ·メニューをよく確認・よく読む。もし半分しかいない状態で食数が多い時は介護担当に申し出ましょう。
  • ·不適切なこと・スタッフ同士の態度・行動は即座に代表へ相談しましょう。
  • ·自分の仕事は責任を持って行いましょう。自分を守るお金や支払いを間違えないようにしましょう。
  • ·ちょっと苦情は大事。ちょっと苦情は大事に稼ぐための大事なことです。はい。チームで働いているので、苦情が分かりで合った場合は不適切な行動はすぐに代表へ相談しましょう。

配膳サービス・食事介助 注意事項

サービス付き高齢者住宅（サ高住）と小規模への配膳

  • ・デイサービス担当へ食事を各々に分ける（カウンターに立って）
  • ・チームごとの名前、目安人数、食事の数を確認してから介護担当に申し出ること
  • ・差し引きは1回行うごとに、チームへ入れ、空へ（左下へ）移動させる

食事形態の確認（必須）

  • ・ソフト食・ミキサー食・普通食は必ずメニューで確認してから盛り付ける
  • ・ゼリー食：ゼリー食はメニューに沿って切って冷蔵庫へ入れる
  • ・柔らかくすべき食材（えのき等）はレシピ量に従って切って冷蔵庫へ入れる
  • ・ご飯は量が人によって多く食べる人がいれば、いる場合に関しては食を少し減らし、11時に残りできるなら、変わり飯をするように
